# 智能温室大棚项目 - KPZ32版本

## 项目概述
基于STM32F103C8T6的智能温室大棚监控系统，使用KPZ32开发板。

## MCU1引脚分配（基于KPZ32开发板）

### 传感器接口
| 传感器 | 引脚 | 说明 |
|--------|------|------|
| DHT11温湿度 | PA1 | 传感器S1接口 |
| TEMT6000光照 | PA0 (ADC) | 传感器S1接口 |
| 土壤湿度 | PA2 (ADC) | 传感器S3接口 |
| SCD40 CO2 | PA15(SDA), PA14(SCL) | 软件I2C |

### PWM舵机接口
| 舵机 | 引脚 | 定时器 |
|------|------|--------|
| 舵机1 | PB3 | TIM2_CH2 |
| 舵机2 | PB8 | TIM4_CH3 |
| 舵机3 | PB9 | TIM4_CH4 |

### 执行器
| 设备 | 引脚 | 说明 |
|------|------|------|
| 继电器/水泵 | PB5 | GPIO输出 |
| 蜂鸣器 | PB2 | GPIO输出 |

### 按键
| 按键 | 引脚 | 功能 |
|------|------|------|
| 按键1 | PA8 | 舵机控制 |
| 按键2 | PA11 | 水泵控制 |

## 功能特性
1. ✅ 实时采集温度、湿度、光照、CO2、土壤湿度数据
2. ✅ 湿度低于阈值时自动开启水泵和蜂鸣器报警
3. ✅ 温度高于阈值时自动开启舵机（通风窗）
4. ✅ 手动按键控制舵机和水泵
5. ✅ 适配KPZ32开发板引脚布局

## 主要变更
相比原始设计，已根据KPZ32开发板进行以下调整：

1. **舵机PWM**：从TIM1改为TIM2和TIM4
   - 舵机1: PA8 → PB3 (TIM2_CH2)
   - 舵机2: PA9 → PB8 (TIM4_CH3)
   - 舵机3: PA10 → PB9 (TIM4_CH4)

2. **传感器引脚**：
   - DHT11: PA0 → PA1
   - TEMT6000: PA1 → PA0 (ADC)
   - 土壤湿度: PA2 (保持不变)

3. **I2C接口**：
   - 从硬件I2C改为软件I2C
   - SDA: PA15, SCL: PA14

4. **执行器**：
   - 继电器: PB12 → PB5
   - 蜂鸣器: PB13 → PB2

5. **按键**：
   - 按键1: PB14 → PA8
   - 按键2: PB15 → PA11

## 使用说明
1. 将传感器连接到对应的KPZ32接口
2. 舵机连接到PWM舵机接口的前3个通道
3. 继电器和蜂鸣器连接到指定GPIO
4. 编译并烧录到STM32F103C8T6
